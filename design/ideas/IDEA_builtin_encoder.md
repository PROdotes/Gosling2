# Built-in Encoder

Stream to Icecast/Shoutcast directly from the app.

## Concept
- Encode audio output to MP3/AAC stream
- Connect to streaming server
- Display listener count, bitrate

## Requirements
- libshout or similar library
- Network connectivity
