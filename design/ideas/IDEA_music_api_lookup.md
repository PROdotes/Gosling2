# Music API Lookup

Fetch metadata from MusicBrainz, Discogs, Spotify API.

## Concept
- Search by artist, title, ISRC
- Auto-fill genre, year, album art
- Multiple API fallbacks

## APIs
- MusicBrainz (free)
- Discogs (free tier)
- Spotify (requires auth)
