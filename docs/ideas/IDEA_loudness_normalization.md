# Loudness Normalization

EBU R128 / ReplayGain loudness normalization.

## Concept
- Analyze track loudness on import
- Store loudness value in database
- Apply gain adjustment during playback
- Consistent volume across all tracks

## Standards
- EBU R128 (broadcast)
- ReplayGain (consumer)
