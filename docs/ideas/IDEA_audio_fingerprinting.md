# Audio Fingerprinting

Auto-identify unknown tracks via acoustic fingerprint.

## Concept
- Generate fingerprint from audio file
- Query services like AcoustID/Shazam
- Auto-fill metadata from match

## Requirements
- Chromaprint or similar library
- Internet connectivity
