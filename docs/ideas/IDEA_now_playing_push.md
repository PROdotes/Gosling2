# Now Playing Push

HTTP POST current track info to website, RDS encoder.

## Concept
- On track change, POST metadata to URL
- Support for RDS (Radio Data System) text
- Webhook integration for websites

## Use Cases
- Update "Now Playing" on website
- Send to RDS encoder for FM display
