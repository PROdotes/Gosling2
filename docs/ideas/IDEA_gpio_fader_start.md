# GPIO / Fader Start

Trigger playback from physical console via GPIO.

## Concept
- Connect to broadcast console's fader start
- GPIO pin triggers play/stop
- Support for Raspberry Pi GPIO or USB relay

## Requirements
- GPIO library
- Physical hardware connection
